## API Used

**TheMealDB API**
A free and open database of meal recipes and ingredients.

* Base URL: `https://www.themealdb.com/api/json/v1/1/`
* Example endpoint:

  ```
  https://www.themealdb.com/api/json/v1/1/filter.php?i=chicken
  ```
